    MIPS指令集

    寄存器文件¶

    PC不在寄存器文件RF中。

    编号 寄存器 功能 编号 寄存器 功能
    0 $0 常量0 24~25 $t8~$t9 临时变量
    1 $at 汇编器保留 26~27 $k0~$k1 操作系统临时变量
    2~3 $v0~$v1 函数返回值 28 $gp 全局指针
    4~7 $a1~$a3 函数参数 29 $sp 堆指针
    8~15 $t0~$t7 临时变量 30 $fp 栈指针
    16~23 $s0~$s7 保存变量 31 $ra 函数返回地址

    B、J指令¶

    PC跳转指令，目的地址计算如下：

    \(JTA(label[25:0]) = \{\ PCplus4[31:28] \ ,\ label \ ,\ 2'b0 \ \}\)

    \(BTA(offset[15:0]) = PCplus4 + (offset << 2)\)

    J型指令 功能 汇编 机器码格式 伪代码
    j 无条件跳转 j target 000010 label PC = JTA
    jal 子程序调用 jal target 000011 label $ra = PC + 4; PC = JTA
    jalr 子程序调用寄存器 jalr $rs, $rd 000000 $rs 00000 00000 00000 001001 $ra = PC + 4; PC = JTA(rs)
    jr 跳转寄存器 jr $rs 000000 $rs 00000 00000 00000 001000 PC = JTA(rs)
    B型指令 功能 汇编 机器码格式 伪代码
    bltz 分支小于零 bltz $rs, label 000001 $rs 00000 offset if ( [rs] < 0) PC =BTA
    bgez 分支大于等于零 bgez $rs, label 000001 $rs 00000 offset if ( [rs] >= 0) PC = BTA
    beq 相等分支 beq $rs, $rt, label 000100 $rs $rt offset if ( [rs] == [rt] ) PC = BTA
    bne 不等分支 bne $rs, $rt, label 000101 $rs $rt offset if ( [rs] != [rt] ) PC = BTA
    blez 分支小于等于零 blez $rs, label 000110 $rs 00000 offset if ( [rs] <= 0) PC = BTA
    bgtz 分支大于零 bgtz $rs, label 000111 $rs 00000 offset if ( [rs] > 0) PC = BTA
    bclf 分支协处理器条件为假 bclf $rs, label 010001 $rs 00000 offset if (!Coprocessor1[$rs] ) PC = BTA
    blct 分支协处理器条件为真 blct $rs, label 010001 $rs 00000 offset if (Coprocessor1[$rs] ) PC = BTA

    R指令¶

    R型指令 功能 汇编 机器码格式 伪代码
    sll 逻辑左移 sll $rd, $rt, sa 000000 00000 $rt $rd sa 000000 [rd] = [rt] << sa
    srl 逻辑右移 srl $rd, $rt, sa 000000 00000 $rt $rd sa 000010 [rd] = [rt] >> sa
    sra 算术右移 sra $rd, $rt, sa 000000 00000 $rt $rd sa 000011 [rd] = [rt] >> sa (算术)
    sllv 逻辑左移变量 sllv $rd, $rt, $rs 000000 $rs $rt $rd 00000 000100 [rd] = [rt] << [rs]
    srlv 逻辑右移变量 srlv $rd, $rt, $rs 000000 $rs $rt $rd 00000 000110 [rd] = [rt] >> [rs]
    srav 算术右移变量 srav $rd, $rt, $rs 000000 $rs $rt $rd 00000 000111 [rd] = [rt] >> [rs] (算术)
    jr 跳转寄存器 jr $rs 000000 $rs 00000 00000 00000 001000 PC = [rs]
    jalr 子程序调用寄存器 jalr $rs, $rd 000000 $rs 00000 $rd 00000 001001 $rd = PC + 4; PC = [rs]
    add 加法 add $rd, $rs, $rt 000000 $rs $rt $rd 00000 100000 [rd] = [rs] + [rt]
    addu 加法（无符号） addu $rd, $rs, $rt 000000 $rs $rt $rd 00000 100001 [rd] = [rs] + [rt] (无符号)
    sub 减法 sub $rd, $rs, $rt 000000 $rs $rt $rd 00000 100010 [rd] = [rs] - [rt]
    subu 减法（无符号） subu $rd, $rs, $rt 000000 $rs $rt $rd 00000 100011 [rd] = [rs] - [rt] (无符号)
    and 与运算 and $rd, $rs, $rt 000000 $rs $rt $rd 00000 100100 [rd] = [rs] & [rt]
    or 或运算 or $rd, $rs, $rt 000000 $rs $rt $rd 00000 100101 [rd] = [rs] | [rt]
    xor 异或运算 xor $rd, $rs, $rt 000000 $rs $rt $rd 00000 100110 [rd] = [rs] ^ [rt]
    nor 非或运算 nor $rd, $rs, $rt 000000 $rs $rt $rd 00000 100111 [rd] = ~( [rs] | [rt] )
    slt 小于比较 slt $rd, $rs, $rt 000000 $rs $rt $rd 00000 101010 [rd] = ( [rs] < [rt] ) ? 1 : 0
    sltu 小于比较（无符号） sltu $rd, $rs, $rt 000000 $rs $rt $rd 00000 101011 [rd] = ( [rs] < [rt] ) ? 1 : 0 (无符号)
    syscall 系统调用 syscall 000000 00000 00000 00000 00000 001100 ExecuteSystemCall()
    break 断点 break code 000000 00000 code code code code 001101 Break(code)
    mfhi 从HI寄存器读取 mfhi $rd 000000 00000 00000 $rd 00000 010000 [rd] = HI
    mthi 向HI寄存器写入 mthi $rs 000000 $rs 00000 00000 00000 010001 HI = [rs]
    mflo 从LO寄存器读取 mflo $rd 000000 00000 00000 $rd 00000 010010 [rd] = LO
    mtlo 向LO寄存器写入 mtlo $rs 000000 $rs 00000 00000 00000 010011 LO = [rs]
    mult 乘法 mult $rs, $rt 000000 $rs $rt 00000 00000 011000 {HI, LO} = [rs] * [rt]
    multu 乘法（无符号） multu $rs, $rt 000000 $rs $rt 00000 00000 011001 {HI, LO} = [rs] * [rt] (无符号)
    div 除法 div $rs, $rt 000000 $rs $rt 00000 00000 011010 LO = [rs] / [rt]; HI = [rs] % [rt]
    divu 除法（无符号） divu $rs, $rt 000000 $rs $rt 00000 00000 011011 LO = [rs] / [rt] (无符号); HI = [rs] % [rt] (无符号)

    I指令¶

    I型指令 功能 汇编 机器码格式 伪代码
    addi 加法立即数 addi $rt, $rs, imm 001000 $rs $rt imm [rt] = [rs] + imm
    addiu 加法立即数（无符号） addiu $rt, $rs, imm 001001 $rs $rt imm [rt] = [rs] + imm (无符号)
    slti 小于比较立即数 slti $rt, $rs, imm 001010 $rs $rt imm [rt] = ( [rs] < imm) ? 1 : 0
    sltiu 小于比较立即数（无符号） sltiu $rt, $rs, imm 001011 $rs $rt imm [rt] = ( [rs] < imm) ? 1 : 0 (无符号)
    andi 与运算立即数 andi $rt, $rs, imm 001100 $rs $rt imm [rt] = [rs] & imm
    ori 或运算立即数 ori $rt, $rs, imm 001101 $rs $rt imm [rt] = [rs] | imm
    xori 异或运算立即数 xori $rt, $rs, imm 001110 $rs $rt imm [rt] = [rs] ^ imm
    lui 装载上位数 lui $rt, imm 001111 00000 $rt imm [rt] = imm << 16
    mul 乘法 mul $rd, $rs, $rt 011100 $rs $rt $rd 00000 000010 [rd] = [rs] * [rt]
    lb 加载字节 lb $rt, offset($base) 100000 $base $rt offset [rt] = SignExtend(Mem[ [base] + offset] )
    lh 加载半字 lh $rt, offset($base) 100001 $base $rt offset [rt] = SignExtend(Mem[ [base] + offset] )
    lw 加载字 lw $rt, offset($base) 100011 $base $rt offset [rt] = Mem[ [base] + offset]
    lbu 加载字节（无符号） lbu $rt, offset($base) 100100 $base $rt offset [rt] = ZeroExtend(Mem[ [base] + offset] )
    lhu 加载半字（无符号） lhu $rt, offset($base) 100101 $base $rt offset [rt] = ZeroExtend(Mem[ [base] + offset] )
    sb 存储字节 sb $rt, offset($base) 101000 $base $rt offset Mem[ [base] + offset] = [rt]
    sh 存储半字 sh $rt, offset($base) 101001 $base $rt offset Mem[ [base] + offset] = [rt]
    sw 存储字 sw $rt, offset($base) 101011 $base $rt offset Mem[ [base] + offset] = [rt]
    lwcl 原子地加载字 lwcl $rt, offset($base) 110000 $base $rt offset [rt] = Mem[ [base] + offset]]
    swcl 原子地存储字 swcl $rt, offset($base) 111000 $base $rt offset mem[ [base] + offset] = [rt]
    mfc0 从协处理器寄存器读取 mfc0 $rt, $rd 010000 $rd 00000 $rt 00000 000000 [rt] = Coprocessor0[$rd]
    mtc0 向协处理器寄存器写入 mtc0 $rt, $rd 010000 $rd 00100 $rt 00000 000000 Coprocessor0[$rd] = [rt]
